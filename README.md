


# 🚀 MERN Stack Task Management System

Ek fully-functional, production-ready Task Management application jise **Isha Gupta** ne develop kiya hai. Yeh app teams ko tasks assign karne, deadlines track karne aur analytics dekhne mein madad karti hai.

## 🌐 Live Demo
*   **Frontend (Live):** (https://task-manager-delta-self.vercel.app/login)
*   **Backend (API):** (https://task-manager-8opj.onrender.com/)

---

## ✨ Key Features
*   🔐 **Role-Based Access Control (RBAC):** Admin aur Member roles ke liye alag-alag permissions.
*   📊 **Interactive Dashboard:** Tasks ka status (To-Do, In-Progress, Done) analytics cards ke sath.
*   🛠 **Project & Task Management:** Projects create karna aur unme multiple tasks assign karna.
*   📅 **Smart Due Dates:** Deadlines track karne ke liye intuitive UI.
*   📱 **Responsive Design:** Mobile, Tablet aur Desktop par perfectly chalta hai.
*   🛡 **Secure Auth:** JWT (JSON Web Tokens) ka use karke secure login aur registration.

---

## 🛠 Tech Stack
**Frontend:**
*   React.js (Vite)
*   Tailwind CSS (Styling ke liye)
*   Axios (API calls ke liye)
*   Lucide React (Icons)

**Backend:**
*   Node.js & Express.js
*   MongoDB Atlas (Database)
*   Mongoose (ODM)
*   JWT & BcryptJS (Security)

**Deployment:**
*   Frontend: **Vercel**
*   Backend: **Render**

## ⚙️ Local Setup Instructions

1.  **Repository Clone karein:**
    ```bash
    git clone (https://github.com/Divyansh21082005/Task-manager.git))
    cd ethara-task-manager
    ```

2.  **Backend Setup:**
    ```bash
    # Backend folder mein jayein
    npm install
    # .env file banayein aur MONGO_URI, JWT_SECRET add karein
    npm start
    ```

3.  **Frontend Setup:**
    ```bash
    cd frontend
    npm install
    npm run dev
    ```
